# Vehicles Web App

Este proyecto consiste en una aplicación web desarrollada con Streamlit que permite explorar y visualizar datos de anuncios de venta de coches.  

## Funcionalidad
- Cargar un conjunto de datos en formato CSV.
- Generar un histograma de la columna "odometer".
- Generar un gráfico de dispersión entre el año del modelo y el odómetro.
- Permitir al usuario seleccionar qué gráficos quiere ver mediante botones o casillas de verificación.
- Visualización interactiva gracias a Plotly.

## Tecnologías
- Python
- Pandas
- Plotly
- Streamlit
